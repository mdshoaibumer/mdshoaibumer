### Engineering Philosophy

- **Correctness first, then performance.** Reliable systems earn trust.
- **Understand the machine.** Memory, syscalls, networking — abstractions leak, and depth matters.
- **Ship, measure, iterate.** Working software in production beats architecture slides.
- **Automate the boring stuff.** If you do it twice, script it. If you script it twice, build a service.
- **Read more code than you write.** The best engineers are relentless readers of systems.
